Detailed Description

A subclass of Machine that represents a weight machine (Bench Press, Preacher Curl, Leg Press).

This class represents a Weight Machine (Bench Press, Preacher Curl, Leg Press) for the gym. It tracks the time that someone started working out on the machine, the weight on the machine, and the number of reps completed.
