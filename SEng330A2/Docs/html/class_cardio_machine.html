Detailed Description

A subclass of Machine that represents a cardio machine (Treadmill, Elliptical, Stationary Bike).

This class represents a Cardio Machine (Treadmill, Elliptical, Stationary Bike) for the gym. It tracks the time that someone started working out on the machine and the resitance level of the machine.
