Detailed Description

A factory class for creating machine objects.

This factory uses the prototype pattern of machines to store a prototype of the two different type of machines. When creating a new machine, it will clone the prototype if it exists or create a new machine and store it for future cloning if it does not. This class is a friend of the Machine, CardioMachine, and WeightMachine classes.

Machine * MachineFactory::createMachine ( const proto::Machine &  m)

Creates a new machine from a protocol buffer machine object.

The new machine will be given a new ID, and default values for startTime (-1), resistance(0), repetitions (0), and weight(0) as appropriate.

The type of this machine can be determined from the proto::Machine object. Will clone an existing machine.
